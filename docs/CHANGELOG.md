## Version 1.2 BETA

### Added
- **Banner UI** - Prominent banner with BISEN branding and version
- **Environment Management** - Full environment system with Dev/Staging/Prod support
- **Export/Import** - BISEN format for sharing collections and environments
- **Request Validation** - JSON/XML validation with syntax highlighting
- **Variable Preview** - Preview variable substitution before sending
- **Enhanced Security** - SSRF protection, rate limiting, input validation
- **Documentation** - Comprehensive documentation in `/docs` folder
- **Author Attribution** - All code files include author information

### Improved
- **UI Design** - Cleaner, more minimalist interface
- **Variable Management** - Better UI for editing and validating variables
- **Response Handling** - Improved JSON formatting and display
- **Error Messages** - More descriptive error messages

### Fixed
- **Database Issues** - SQLite database initialization
- **Variable Substitution** - Fixed variable replacement logic
- **Collection Navigation** - Improved sidebar expand/collapse
